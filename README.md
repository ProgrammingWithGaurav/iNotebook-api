## iNotebook-CRUD-API
- this api i used in my MERN-iNotebook repository
- you can make as many request to this api 
- This api is hosted in heroku link: http://my-inotebook-api.herokuapp.com/

#### MADE with :
- mongoose
- nodejs
- express

### ROUTES THIS API CONTAIN
#### FOR USER
- /login
- /signup
- /deleteuser
- /getuser

#### FOR NOTES
- /addnote
- /deletenote
- /updatenote
